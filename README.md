This is a responsive personal portfolio website built with **React** and **Tailwind CSS**.  
It showcases my profile, skills, certifications, experience, and contact information as a budding Network Engineer and Computer Science graduate.

## Features

- Clean and modern UI with Tailwind CSS
- Sections:
  - Hero (with profile image)
  - About Me
  - Skills
  - Work Experience
  - Certifications
  - Contact

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start development server
```bash
npm start
```

## Add Your Profile Picture

Place your profile image as `profile.jpg` inside the `src/assets/` folder.
